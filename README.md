# **Installatie**
**1. Clone de repository:**  
```git clone https://github.com/jouw-username/flofonesite.git```  
**2. Navigeer naar de directory:**  
```cd flofonesite```  
**3. Installeer de vereiste pakketten:**   
```npm install```  
**4. Start de server:**   
```npm start```
